# Grocery Store Management System (SQL)

## Project Overview
This project is designed to manage grocery store operations using SQL. It helps store customer, product, order, and sales information efficiently.

## Technologies Used
- SQL
- MySQL

## Features
- Customer Management
- Product Management
- Order Tracking
- Sales Analysis
- Inventory Management

## SQL Concepts Used
- SELECT, WHERE, ORDER BY
- GROUP BY, HAVING
- JOINS (INNER JOIN, LEFT JOIN)
- Subqueries
- Aggregate Functions
- Views

## Database Tables
- Customers
- Products
- Orders
- Order_Details

## Sample Analysis
- Total sales by product
- Top-selling products
- Customer purchase history
- Monthly sales reports
- Inventory status

## Outcome
Developed SQL queries to manage and analyze grocery store data, helping generate business insights and improve decision-making.
